# HTML Registration Form

My first HTML project — built as part of my Full Stack Web Development journey.

## Features

- Full Name, Email, Password, Confirm Password
- Phone Number and Date of Birth
- Gender Selection (Radio Buttons)
- Course and Country Dropdowns
- Skills Checkboxes
- Message Textarea
- Profile Picture Upload
- Submit and Reset Buttons
- Required attribute validation

## Technologies Used

- HTML5 only — no CSS, no JavaScript

## What I Practiced

- HTML Document Structure and Boilerplate
- Form tag and all major input types
- Labels with proper for and id connection
- Radio Buttons, Checkboxes, Select Dropdowns
- Textarea and File Input
- Required attribute to prevent empty submission
